GSL Cloud is the corporate and admin dashboard for GoSpeedLog, designed to handle complex logistics operations. It features seamless integration with multiple carriers and e-commerce platforms, providing a unified solution for shipment management.

### Key Features

- **Multi-Carrier Integration**: Supports UPS, Canada Post, UniUni, USPS, EasyPost, and more.
- **E-commerce Connectivity**: Links directly with e-commerce merchants for automated order processing.
- **Corporate Dashboard**: A comprehensive view for admin users to manage tenants, fleets, and global settings.

### Tech Stack

- **Backend**: Elixir, Phoenix Framework
- **Frontend**: Phoenix LiveView, Tailwind CSS
- **Database**: PostgreSQL
- **Infrastructure**: Docker-based deployment

This platform empowers logistics companies to scale their operations efficiently while maintaining high visibility and control over their supply chain.
